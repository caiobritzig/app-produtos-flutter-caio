## 📱 Descrição do Projeto

Aplicativo de cadastro de produtos desenvolvido em Flutter como parte da atividade acadêmica.
O objetivo foi praticar o uso de **StatefulWidget**, navegação entre telas, passagem de dados e organização do projeto em camadas (**models** e **screens**).

O aplicativo permite cadastrar produtos, visualizar a lista e acessar os detalhes de cada item.

---

## ✅ Funcionalidades Implementadas

### Obrigatórias

* Cadastro de produtos com formulário
* Exibição de lista de produtos com ListView
* Navegação entre telas com `Navigator.push()`
* Visualização de detalhes de cada produto
* Retorno de dados entre telas

### ⭐ Funcionalidades Extras

* Validação básica dos campos no cadastro
* Organização do projeto em pastas (models e screens)
* Interface seguindo padrões do Material Design
